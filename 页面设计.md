# 页面设计

1. ## 首页 / 算法广场页面

   白色极简商务风网页UI设计，全屏纯白底色，低饱和度浅灰色作为辅助色，无复杂渐变、无花哨装饰，留白充足，扁平化卡片设计，圆角8px，细线条分割，现代无衬线字体，信息层级清晰，干净专业，偏向程序员教学工具风格，高对比度保证代码文字可读性，响应式网页布局，无多余动效，仅保留轻微hover交互反馈，整体冷静理性、学术简约，适合高校算法教学场景。

2. ## 算法在线可视化运行页（核心页面）

   白色简约算法可视化调试页面，左右双栏布局：左侧代码编辑区域，浅灰行号底色，代码高亮清晰；右侧分为上下两块，上方算法动态执行可视化画布，白色背景+灰色细连接线展示数据状态变化，下方执行步骤表格，每一步记录程序快照、步骤说明；顶部放置输入数据选择、自定义输入、运行、暂停、重置按钮，按钮浅灰色极简样式，执行状态用极简标签展示（运行中/完成/异常），整体界面理性克制，突出算法动态演示效果，留白充足，保证数据快照JSON内容可读性。

3. ## 高级风格

   偏向LeetCode极简网页UI风格，功能优先，视觉克制，注重代码、表格、数据可视化可读性，少量蓝色作为功能强调色。

4. ## 要求

   禁止深色模式、禁止渐变背景、禁止3D立体效果、禁止插画表情包、禁止炫酷光影特效、禁止高饱和艳丽配色，仅允许白色+不同明度灰色作为主辅色，严格遵循简约学术风，网页端UI界面，不要移动端APP样式。
